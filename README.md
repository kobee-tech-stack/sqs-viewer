# SQS Viewer
### Node.js AWS SQS Viewer - Monitor & Execute Trigger A Message For Testing

## How to run it
### Step 1: init & install dependencies

```shell
yarn install 
```

### Step 2: Trigger a build & run
```shell
yarn build
yarn start
```
